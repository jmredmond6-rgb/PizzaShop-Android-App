# PizzaShop Android Project -- Development Context

## Overview

PizzaShop is an Android mobile application developed in Java using Android Studio. The application simulates a pizza ordering workflow including authentication, menu browsing, cart management, checkout, and order history.

The project was originally created as a software engineering capstone project and later refined into a professional GitHub portfolio application.

## Technology Stack

Language - Java

Android Framework - Android Studio - RecyclerView - Material Components - Room Persistence Library

Database - SQLite via Room ORM

Security - PBKDF2 password hashing for local credential storage

Development Tools - Git - GitHub - Gradle

## Architecture Pattern

UI Layer -> Repository -> DAO -> Room Database -> Entities
